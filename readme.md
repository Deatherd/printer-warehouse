# 🖨️ Printer Warehouse - Система учета расходных материалов для принтеров

Веб-приложение для учета и управления расходными материалами офисных принтеров: картриджи, фьюзеры, тонеры и другие расходники.

## ✨ Возможности

- 📊 **Дашборд** - визуальная статистика остатков и предупреждения о низких запасах
- ➕ **Добавление на склад** - пополнение запасов расходных материалов
- ➖ **Списание** - учет расхода материалов с указанием причины
- 📦 **Управление позициями** - добавление новых товаров и категорий
- 📝 **Логирование** - полная история всех операций с фильтрацией
- 🔐 **Авторизация** - защищенный доступ по логину/паролю
- 🚨 **Уведомления** - автоматическое оповещение о заканчивающихся позициях

## 🛠️ Технологии

- **Backend**: Python 3.11 + Flask
- **База данных**: SQLite
- **Аутентификация**: Flask-Login
- **Frontend**: Bootstrap 5, Jinja2
- **Деплой**: Docker, Docker Compose, Nginx (опционально)
